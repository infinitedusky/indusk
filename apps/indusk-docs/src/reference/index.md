# Reference

Reference documentation for InDusk skills and tools.

## Skills

The development lifecycle is powered by six skills that work together:

| Skill | Purpose |
|-------|---------|
| [Plan](/reference/skills/plan) | Structured planning lifecycle |
| [Work](/reference/skills/work) | Execute impl checklists |
| [Verify](/reference/skills/verify) | Automated checks per phase |
| [Context](/reference/skills/context) | Living project memory |
| [Document](/reference/skills/document) | Per-phase documentation gate |
| [Retrospective](/reference/skills/retrospective) | Closing audit and knowledge handoff |

## Tools

| Tool | Purpose |
|------|---------|
| [Composable.env](/reference/tools/composable-env) | Docker-based local dev environment |
| [CodeGraphContext](/reference/tools/codegraph) | Structural code intelligence |
